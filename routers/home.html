Welcome to CodeBacklog API!

CodeBacklog is an API used by Competitive programmers , DSA question practisers, you can store unsolve or important questions with notes and it will remind you to have a look in future. 

How to use:

first of all you have to make an account. then after you can trigger API calls.and use these service.

Some useful features:

  1. Interactive source editor
  2. Clean Interface and Easy to use
  3. Remainder service 

             

Cleaning options:

Name of API : request mode  Path need token
Register: post /users/sign_up  
login: post /users/sign_in  
auther:get  /users/me yes
allusers: get /users  
user by id : get /users/:id  
all problems: get /problems  
problem by id /problems/:id  
auther's problems: get /problems/me yes
create problem : post /problems yes
update problem: put /problems/:id yes
delete problems:delete /problems/:id yes
